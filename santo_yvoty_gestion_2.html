⚙️ Configuración inicial — Google Drive

Para guardar los datos en la nube necesitás crear credenciales gratuitas en Google Cloud (5 minutos).

  1. Andá a console.cloud.google.com e iniciá sesión
  2. Creá un proyecto nuevo (o usá uno existente)
  3. Habilitá la Google Drive API (Biblioteca → buscar "Drive API")
  4. Creá credenciales: ID de cliente OAuth 2.0 → Tipo: Aplicación web
  5. En "Orígenes autorizados" agregá la URL donde abrís este archivo (o http://localhost)
  6. Creá también una Clave de API
  7. Pegá ambos valores abajo y guardá
